# Recipe Website

The Recipe Website is a user-friendly platform dedicated to sharing recipes from around the world. This repository contains the source code for the main page (home page), the recipe page, the about us page, and the contact page.

## Technologies Used

- **HTML**: The markup language used for structuring the content on the website.
- **CSS**: The style-sheet language used for describing the look and formatting of the document written in HTML.
- **JavaScript**: The programming language used to create dynamic and interactive elements on the website.

## Libraries and Frameworks

- **jQuery**: A fast, small, and feature-rich JavaScript library, used in this project for DOM manipulation and AJAX calls.
- **jQuery UI**: Provides abstractions for low-level interaction and animation, advanced effects and high-level, themeable widgets. In this project, it's used to create dialog popups and enhance form functionality.

## Common Features

- **Navigation Bar**: Allows users to navigate between different pages of the website.
- **Footer Section**: Contains the site's contact information and links to various social media pages.

### Main Page

#### Technologies Used

- **Font Awesome**: A font and icon toolkit based on CSS and Less, used in this website for creating attractive and intuitive icons.
- **Lightbox**: A JavaScript library that displays images and videos by filling the screen and dimming out the rest of the webpage. Used in the website to view the images of recipes in a user-friendly manner.

#### Features

- **Lightbox for Images**: Provides an engaging and user-friendly way to view recipe images.
- **Icons from Font Awesome**: Enhances the visual appeal and user intuitiveness of the website.

### Recipe Page

#### Technologies Used

- **Filterizr**: A jQuery plugin that sorts and filters items in a grid layout, in this case, the recipe cards.
- **jQuery UI Dialog**: Sets up a dialog box to appear when the "View Recipe" button on each recipe card is clicked.

#### Features

- **Recipe Section**: Displays recipe cards with information such as the recipe name, and a brief description. Users can filter these recipes by category with the help of Filterizr.
- **Fetch Recipes Button**: Fetches a random recipe from the TheMealDB API and dynamically adds a new recipe card to the page.
- **Recipe Dialogs**: Each 'View Recipe' button opens a dialog (jQuery UI dialog) with more detailed information about the respective recipe.

### Contact Us Page

#### Technologies Used

- **jQuery UI**: This library provides abstractions for low-level interaction and animation, advanced effects and high-level, themeable widgets. Used in this project to implement the datepicker and autocomplete features in the form.

#### Features

- **Date Picker**: Provides an interactive calendar that allows users to select their birth dates in a user-friendly manner.
- **Autocomplete**: Suggests and fills in the country name as the user begins to type it in the 'Country' field.

### About Us Page

Provides an overview of the Recipe Website team and its mission.

## Usage

1. Clone or download this repository.
2. Open `index.html` in your web browser.
3. Explore the website, view popular recipes, and search for your desired recipes.
4. Navigate to different pages using the navigation bar at the top.
5. Connect via the social media links in the footer section.
